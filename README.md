## SAUCEDEMO

PROJETO DE AUTOMAÇÃO DE TESTES USANDO O CYPRESS.
ESSE PROJETO AUTOMATIZA O SITE https://www.saucedemo.com. CONSISTE EM UM SITE DE COMÉRCIO ONDE VOCÊ FAZ LOGIN, ADICIONA OS PRODUTOS NO CARRINHO E FINALIZA A COMPRA.

**************************
PARA O RODAR O PROJETO, É NECESSÁRIO CUMPRIR OS SEGUINTES PRÉ-REQUISITOS:

1. Node.js (https://nodejs.org)
2. NPM ou Yarn - instalado junto ao node.js
3. VS Code (https://visualstudio.microsoft.com/pt-br/downloads/)
4. Git bash
5. Navegador compatível (Chrome)

**************************
Para instalação do cypress

1. Rodar o comando: npm install - instalação das dependências do projeto
2. Executar e abrir a interface cypress: npx cypress open
3. Executar o cypress em modo headless: npx cypress run
